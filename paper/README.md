# Paper: Mapping the Narrow Corridor with LLMs

Source for the arXiv paper. Self-contained two-column preprint (`\documentclass[twocolumn]{article}` — no external `arxiv.sty` needed).

```
paper/
  main.tex            # the paper
  references.bib      # related work
  tables/*.tex        # generated by analysis.py (\input into main.tex)
  experiments/
    run_experiments.py  # 8 countries x 4 models -> results/*.{json,png,gif}
    ensemble.py         # mean-across-models atlas (Fig. 2)
    atlas_combined.py   # all countries in one shared space (Fig. 3)
    analysis.py         # results/ + external CSV -> tables/*.tex
    runs/               # generated (gitignored)
    external/vdem.csv   # you provide (see below)
```

## Build the PDF

```bash
cd paper
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

It compiles **before** experiments are run: missing atlas panels render as
labeled placeholder boxes and the tables ship with placeholder rows. Everything
still-to-fill is shown in red via the `\ph{...}` macro — grep `main.tex` for `\ph`
to find open items.

## Reproduce the experiments

> Full step-by-step plan — commands, data processing, tables/figures, and how to
> interpret each result — is in **[RUNBOOK.md](RUNBOOK.md)**. Quick version below.

Needs provider API keys in the repo-root `.env` (see `../.env.example`):
`GEMINI_API_KEY`, `ANTHROPIC_API_KEY`, `OPENAI_API_KEY`, `OPENROUTER_API_KEY`.

```bash
# from the repo root
uv run python paper/experiments/run_experiments.py --gif   # JSON + PNG (+ GIF)
uv run python paper/experiments/analysis.py                # writes tables/*.tex
uv run python paper/experiments/summarize.py               # derived/*.csv for the prose
```

`run_experiments.py` is idempotent — responses are cached under `runs/.cache/`,
so interrupted sweeps resume and re-runs are free. Countries, models, period
step, and end year are configured at the top of `run_experiments.py`.

## External validation data (you provide)

`analysis.py` correlates LLM scores with an expert-index CSV at
`paper/experiments/external/vdem.csv`. It is **not** bundled (size/licensing).

- **V-Dem** (recommended): download the dataset CSV from
  <https://www.v-dem.org/data/the-v-dem-dataset/>. It already has `country_name`
  and `year` columns plus the indicators.
- Default indicator columns (edit the `EXT_*` constants in `analysis.py` to
  taste): society ← `v2xcs_ccsi` (core civil society index); state ← `v2svstterr`
  (state authority over territory). `v2svstterr` is a narrow proxy; consider
  `v2clrspct` (impartial public administration) or an external state-capacity
  dataset (e.g. Hanson–Sigman) and re-running.
- The V-Dem table reports Spearman correlations on both levels and first
  differences; inter-model agreement uses Krippendorff's alpha (interval) on
  first differences. Polity5 / Freedom House can be added by pointing the
  constants at their columns.

Without the CSV the validation table is written with `\ph{--}` placeholders so
the paper still compiles.
